# Terrier
IDEA Plugin for Android Studio: show activity stack ; send text to EditText.

![Showcase](/screen.png)



## Usage

* show activity stack (adb shell dumpsys activity activities)*

* make Android EditText in focus, enter text in plugin and click send buttonï¼Œ text will be entered in android EditText *
